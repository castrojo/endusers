The CNCF End User Community gathers at KubeCon + CloudNativeCon for focused events, conversations, and networking.

## End User Summit

The **End User Summit** brings together cloud native practitioners from organizations using cloud native technologies in production. It is a forum for sharing operational experience, architecture lessons, and practical guidance with peers.

- [End User Summit — KubeCon + CloudNativeCon North America 2025](https://events.linuxfoundation.org/archive/2025/kubecon-cloudnativecon-north-america/)

## End User Reception

The **End User Reception** is a networking event for members of the CNCF End User Community during KubeCon + CloudNativeCon.

- [KubeCon + CloudNativeCon North America 2025 event resources](https://events.linuxfoundation.org/archive/2025/kubecon-cloudnativecon-north-america/attend/event-resources/)

> Event pages and schedules are maintained by the Linux Foundation. Links above point to the latest KubeCon + CloudNativeCon North America materials.
